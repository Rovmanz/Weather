# 🌦️ Weather — прогноз погоды в консоли

Консольное Python-приложение для просмотра прогноза погоды по городу: на день (почасово), на неделю и на ближайшие дни (псевдо-«месяц»). Данные берутся из бесплатного API [Open-Meteo](https://open-meteo.com/) — без регистрации и API-ключа.

## Возможности

- 🔎 Поиск города по названию (геокодер Open-Meteo)
- ⏰ Погода на сегодня **по часам**
- 📆 Погода на **неделю** (7 дней) с мин/макс температурой
- 🗓️ Погода на **ближайшие дни** (максимум 16 дней — ограничение бесплатного API)
- 😀 Эмодзи для каждого состояния погоды (дождь, снег, гроза, туман и т.д.)

## Требования

- Python 3.8+
- Библиотека `requests`

## Установка

```bash
pip install requests
```

## Запуск


## Ограничения

- Бесплатный тариф Open-Meteo отдаёт прогноз максимум на **16 дней** вперёд, поэтому полноценного прогноза на месяц не существует — выводятся все доступные дни.
- Названия городов лучше вводить на английском или в общепринятом написании — это повышает точность поиска геокодером.
